# flooding_system
Adalah program yang menjalankan sistem pendeteksi banjir menggunakan sensor ultrasonik yang kemudian datanya akan di proses di server. Hasilnya akan berupa pemberitahuan dini terhadap bencana banjir yang akan datang melalui Whatsapp.

| Field         | Isi             |
| ------------- | --------------- |
| Name          | flooding-system |
| Runtime       | Node            |
| Build Command | `npm install`   |
| Start Command | `npm start`     |
